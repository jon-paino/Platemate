Platemate Backend Documentation

This documentation provides an overview of the `app.py` module in the Platemate Backend application.

Module Overview

The `app.py` module defines the main Flask application for the Platemate backend. It includes endpoints for uploading images and generating equipment recommendations.

Functions

allowed_file

Check if the uploaded file has a valid extension.

  • Args:
    • filename (str): Name of the file to check.
  • Returns:
    • bool: True if the file extension is allowed, False otherwise.

Endpoints

/upload-images

Method: POST

Description: Endpoint to handle image uploads. Receives image files, processes them to extract equipment information, and returns the results.

Returns: JSON: A JSON response containing equipment information or an error message.

/get-equipment-recommendations

Method: POST

Description: Endpoint to generate additional equipment recommendations. Receives a list of current equipment, generates recommendations using an external API, and returns the results.

Returns: JSON: A JSON response containing additional equipment recommendations or an error message.

Entry Point

Function: `__main__`

Description: Entry point for running the Flask application. The application runs in debug mode for development purposes.
